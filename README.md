# Heading Numberer

这是一个 Obsidian 插件，用于自动为笔记中的标题添加编号。

## 功能简介
- 自动为 Markdown 标题（如 #、##、### 等）添加递增编号。
- 支持多级标题编号。
- 可自定义编号格式。

## 安装方法
1. 下载本插件文件。
2. 将插件文件放入 Obsidian 的 `.obsidian/plugins/obsidian-heading-numberer` 文件夹。
3. 在 Obsidian 设置中启用本插件。

## 使用说明
启用插件后，编辑笔记时标题会自动添加编号，无需额外操作。

## 文件结构
- `main.js`：插件主逻辑
- `manifest.json`：插件元数据
- `data.json`：插件数据存储（如有）
